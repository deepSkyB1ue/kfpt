数据名称：

历史轨迹数据，用于在GIS系统中显示车辆的历史轨迹，包括：点位标识、图标编号、历史定位数据等。

接口函数：

SetData

数据类型代码：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="HisLocation">

<LName Type = "TEXT">用来显示轨迹线的图层名称</LName>

<PData>

<CallBack Type = "TEXT">是否需要回调（1：是；0：否）</CallBack>

<PageStart Type="TEXT">是否是起始页（1：是；0：否）</PageStart>

<PageEnd Type="TEXT">是否已到末页（1：是；0：否）</PageEnd>

<StartIcon Type = "TEXT">起点图标ID或相对路径</ StartIcon>

<Icon Type = "TEXT">图标ID或相对路径</Icon>

</PData>

<PLocation>

<Guid Type = "GUID">点位ID</Guid>

<Label Type = "TEXT">点位显示名称</Label>

<LON Type = "SINGLE">经度</LON>

<LAT Type = "SINGLE">纬度</LAT>

<Time Type = "DATE">时间</Time>

<Speed Type = " SINGLE">速度</Speed>

<Direction Type = "LONG">方向</Direction>

<Status Type=”TEXT”>状态</Status>

<Icon Type = "TEXT">图标ID或相对路径</Icon>

<Angle Type = "TEXT">角度（0度为正北方向，可选，默认为0）</Angle>

</PLocation>

…（数据可重复多次出现）

<LSymbol>

<LColor Type = "TEXT">轨迹线颜色（#RGB，例：#00FF00）</LColor>

<LStyle Type = "TEXT">轨迹线线型，dash：虚线；solid：实线 </LStyle>

<LWight Type = " SINGLE ">线宽</LWight>

<Transpatecy Type = " SINGLE ">透明度（例：0.8）</Transpatecy>

</LSymbol>

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示轨迹回放的图层名称。
PageStart Text 是
是否是第一页。
0：否；
1：是；
PageEnd Text 是
是否是最后一页。
0：否；
1：是；
Guid GUID 是 点位信息的唯一标识（Guid）
Label TEXT 否 点位显示的名称
LON SINGLE 是 经度举例：“120.123456”
LAT SINGLE 是 纬度举例：“31.123456”
Time DATE 否 定位时间
Speed SINGLE 否 当前速度(Km/h)
Direction LONG 否 方向
Status TEXT 否 状态
Icon TEXT 是 图标编号或相对路径（BS相对路径为图标的URL，CS相对路径为，相对于GIS组件所在目录中指定图片文件夹GIMapImage的路径）
Angle SINGLE 否 图标角度，0度为正北，默认为0
LColor TEXT 否 轨迹线颜色RGB，如#00FF00
LStyle TEXT 否 线形，dash虚线，solid实线
LWeight SINGLE 否 线宽
Transpatecy SINGLE 否 透明度

数据完整性说明：

点位标识是主要标识；

图层名称是指定轨迹回放显示的图层，不能为空；

是否为第一页和是否为最后一页不能为空；

经度、纬度不能为空；

车辆数据被删除后，车辆定位数据也将同步删除。

流转方式：

轨迹回放数据，由业务系统产生，不在其他系统中流转。

数据生成的频度：

根据应用需求产生。

设置轨迹回放显示方式

数据名称：

轨迹回放的显示方式。

接口函数：

SetData

数据类型代码：

HisLocationDisplay。

数据定义：

轨迹回放时设置显示轨迹点或轨迹线。

数据格式：

<Document TaskGuid = "组件标识" DataGuid = "数据标识" DataType = "HisLocationDisplay">

<IsPoint Type = "Text">是否只显示轨迹点（1：是；0：否）</IsPoint>

</Document>

数据描述：

节点名称 类型 是否必填 含义
IsPoint Text 是 是否只显示轨迹点（1：是；0：否）

流转方式：

消息由GIS系统产生，反馈给业务系统。

访问权限：

授权系统前台页面可获取该信息；

其他未授权系统无法进行获取；

数据生成的频度：

轨迹回放时根据应用需求产生。

设置取消或暂停轨迹回放

数据名称：

取消或暂停GIS地图中的轨迹回放。

接口函数：

SetData

数据类型代码：

CancelHisLocation

数据定义：

取消或暂停轨迹回放。

数据格式：

<Document TaskGuid = "组件标识" DataGuid = "数据标识" DataType = "CancelHisLocation">

<Cancel = " TEXT ">0/1/2</Cancel>

<AutoTrack = " TEXT ">0/1</AutoTrack>

</Document>

数据描述：

节点名称 类型 是否必填 含义
Cancel TEXT 是
是否结束轨迹回放。
0：暂停；
1：停止；
2：继续回放；
AutoTrack TEXT 是
是否跟踪轨迹回放。
0：跟踪；
1：不跟踪；

流转方式：

消息由业务系统产生，反馈给GIS系统。

访问权限：

授权系统前台页面可获取该信息；

其他未授权系统无法进行获取；

数据生成的频度：

根据应用需求产生。

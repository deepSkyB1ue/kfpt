添加地图对象

将此代码添加到需要嵌入地图的页面中，修改地图应用服务地址，即可访问地图服务。格式如下：

<div id="szMapContainer" style='width: 100%; height: 100%;'>

<object id="_szmap" classid="clsid:D27CDB6E-AE6D-11cf-96B8-444553540000" width="100%" height="100%">

<param name="movie" value="地图调用地址" />

<param name="base" value="地图项目地址" />

<param name="bgcolor" value="#FFFFFF" />

<param name="allowScriptAccess" value="always" />

<param name="allowFullScreen" value="true" />

<param name="wmode" value="window"/>

<param name="flashvars" value="config=config.xml" />

<object id="_szmap2" type="application/x-shockwave-flash" data="地图调用地址" width="100%" height="100%">

<param name="bgcolor" value="#FFFFFF" />

<param name="base" value="地图项目地址" />

<param name="allowScriptAccess" value="always" />

<param name="allowFullScreen" value="true" />

<param name="flashvars" value="config=config.xml" />

</object>

</object>

</div>

地图调用地址例如：http://服务器IP/xzpgis/index.swf

地图项目地址例如：http:// 服务器IP /xzpgis/

引入JS文件

将interface.js接口文件拷贝至所需项目中，并在需要调用接口的页面中引入此JS文件，即可使用此接口中的方法调用地图服务。

<script src="JS文件路径" type="text/javascript"></script>

调用接口示例

方法名:

SetData

参数:

userId：用户ID

taskGuid：组件标识

dataGuid：图层标识

type：数据类型代码（如：AddLayer）

xmlData：数据

（如：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="AddLayer">

<LName Type = "Text">图层名</LName>

<LType Type = "Text">图层类型（Point、Line、Polygon）</LName>

…（数据可重复多次出现）

</Document>

）

数据名称：

子图源信息，包括：子图源服务路径、显示比例尺范围设定。

数据类型代码：

AddMapServer

数据定义：

图层名称、自动缩放、自动居中、面元素GUID、点集、颜色、透明度。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="AddMapServer">

<LName Type = "TEXT">用来显示子图源的图层名称</LName>

<AutoZoom Type =”TEXT<是否自动缩放1：是，0：否（可选，默认为1）>/AutoZoom>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<Data>

<MName Type = "TEXT">szmap</MName>

<MServer Type = "TEXT">http://gisserver.szjt.cn:8092/arcgis/services</MServer>

<MType Type = "TEXT"<服务类型（tiled或dynamic，默认：dynamic）</MType>

<MMinScale Type = "TEXT">比例尺(例如：20001)</MMinScale>(定义的比例尺范围，不能超过地图底图的比例尺范围)

<MMaxScale Type = "TEXT">例尺(例如：1000000)>/MMaxScale>

</Data>

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示面元素的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
MName TEXT 是 线段信息的唯一标识（Guid）
MServer TEXT 是 服务地址
MMinScale TEXT 是 比例尺
MMaxScale TEXT 是 比例尺

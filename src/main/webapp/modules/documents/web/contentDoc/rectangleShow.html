数据名称：

矩形元素数据信息，维护矩形元素数据的基本信息，包括：颜色、透明度设定。

数据类型代码：

DrawRectangle

数据定义：

图层名称、自动缩放、自动居中、最小点、最大点、颜色、透明度。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType=" DrawRectangle ">

<LName Type = "TEXT">用来显示矩形的图层名称</LName>

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<Transpatecy Type = " SINGLE >透明度（例：0.8，可选，默认为1）</Transpatecy>

<RData>

<Guid Type=”GUID”>面ID</Guid>

<LTLON Type="SINGLE">左上角经度</LTLON>

<LTLAT Type="SINGLE">左上角纬度</LTLAT>

<RDLON Type="SINGLE">右下角经度</RDLON>

<RDLAT Type="SINGLE">右下角纬度</RDLAT>

<RColor Type = "TEXT">矩形颜色（#RGB，例：#00FF00，可选，默认为#FF0000）</RColor>

</RData>

…（数据可重复多次出现）

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示矩形元素的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
Guid GUID 是 矩形元素信息的唯一标识（Guid）
LTLON SINGLE 是 左上角经度
LTLAT SINGLE 是 左上角纬度
RDLON SINGLE 是 右下角经度
RDLAT SINGLE 是 右下角纬度
RColor TEXT 否 矩形元素颜色，颜色RGB，如#00FF00，默认为：#FF0000。
Transpatecy SINGLE 否 透明度，值域范围：0 -----100;默认为：100

回调函数为了与接口函数相区分，命名规则统一为接口函数名后加“_CallBack”进行命名，如“SetData”的回调函数为“SetData_CallBack”，函数参数分为JavaScript和.Net两种方式：

JavaScript的与接口函数相同，如SetData_CallBack ( UserID As string ,  TaskGuid As string ,  DataGuid As string ,  DataType As string ,  XmlData As string ) As int。

.Net方式，采用事件机制，如SetData_Callback(this, new SetDataEventArgs(UserID, TaskGuid, DataGuid, DataType, XmlData))，其中SetDataEventArgs派生于EventArgs类：

public class SetDataEventArgs : EventArgs

{

public SetDataEventArgs(string UserID, string TaskGuid, string DataGuid, string DataType, string XmlData)

{

this.UserID = UserID;

this.TaskGuid = TaskGuid;

this.DataGuid = DataGuid;

this.DataType = DataType;

this.XmlData = XmlData;

}

public string UserID;

public string TaskGuid;

public string DataGuid;

public string DataType;

public string XmlData;

}

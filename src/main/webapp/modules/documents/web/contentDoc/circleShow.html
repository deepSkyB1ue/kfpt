数据名称：

圆元素数据信息，维护面元素数据的基本信息，包括：颜色、透明度设定。

数据类型代码：

DrawCircle

数据定义：

图层名称、自动缩放、自动居中、圆心、半径、颜色、透明度。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="DrawCircle">

<LName Type = "TEXT">用来显示面的图层名称</LName>

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<Transpatecy Type = " SINGLE ">透明度（例：0.8，可选，默认为1）</Transpatecy>

<CData>

<Guid Type=”GUID”>面ID</Guid>

<CenterPoint Type = "TEXT">120.123456,31.123456</ CenterPoint >

<Radius Type = "TEXT">10000</ Radius >

<CColor Type = "TEXT">面颜色（#RGB，例：#00FF00，可选，默认为#FF0000）</CColor>

</CData>

…（数据可重复多次出现）

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示圆元素的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
Guid GUID 是 面元素信息的唯一标识（Guid）
CenterPoint TEXT 是 圆心
Radius TEXT 是 面元素颜色，颜色RGB，如#00FF00，默认为：#FF0000。
Transpatecy SINGLE 否 透明度，值域范围：0 -----100;默认为：100

数据名称：

面元素数据信息，维护面元素数据的基本信息，包括：颜色、透明度设定。

数据类型代码：

DrawPolygon

数据定义：

图层名称、自动缩放、自动居中、面元素GUID、点集、颜色、透明度。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="DrawPolygon">

<LName Type = "TEXT">用来显示面的图层名称</LName>

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<Transpatecy Type = " SINGLE ">透明度（例：0.8，可选，默认为100）</Transpatecy>

<AData>g

<Guid Type=”GUID”>面ID</Guid>

<POINTS Type = "TEXT">点集（例：120.123456,31.123456;120.654321,31.654321）</POINTS>

<AColor Type = "TEXT">面颜色（#RGB，例：#00FF00，可选，默认为#FF0000）</AColor>

</AData>

…（数据可重复多次出现）

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示面元素的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
Guid GUID 是 面元素信息的唯一标识（Guid）
POINTS TEXT 是 线段点位集合，以逗号（,）区分经度、纬度；以分号（;）分隔点位。
AColor TEXT 否 面元素颜色，颜色RGB，如#00FF00，默认为：#FF0000。
Transpatecy SINGLE 否 透明度，值域范围：0 -----100;默认为：100

数据完整性说明：

面元素标识是主要标识；

图层名称指定线段所在图层（业务系统维护）；

流转方式：

由业务系统产生，不在其他系统中流转。

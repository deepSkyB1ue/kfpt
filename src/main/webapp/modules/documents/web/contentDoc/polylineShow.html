数据名称：

矩形元素数据信息，维护矩形元素数据的基本信息，包括：颜色、透明度设定。

数据类型代码：

DrawPolyline

数据定义：

图层名称、自动缩放、自动居中、线段GUID、点集、样式。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="DrawPolyline">

<LName Type = "TEXT">用来显示线段的图层名称</LName>

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>

<AutoPan Type =”TEXT>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<Transpatecy Type = " SINGLE ">透明度（例：0.8，可选，默认为100）</Transpatecy>

<LData>

<Guid Type=”GUID”>线段ID</Guid>

<POINTS Type = "TEXT">点集（例：120.123456,31.123456;120.654321,31.654321）</POINTS>

<LColor Type = "TEXT">线段颜色（#RGB，例：#00FF00，可选，默认为#FF0000）</LColor>

<LStyle Type = "TEXT">线型，dash：虚线；solid：实线（可选，默认为solid）</LStyle>

<LWight Type = " SINGLE ">线宽（可选，默认为1）</LWight>

</LData>

…（数据可重复多次出现）

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示线段的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
Guid GUID 是 线段信息的唯一标识（Guid）
POINTS TEXT 是 线段点位集合，以逗号（,）区分经度、纬度；以分号（;）分隔点位。
LColor TEXT 否 线段颜色，颜色RGB，如#00FF00，默认为：#FF0000。
LStyle TEXT 否 线型，dash虚线，solid实线，默认为：solid
LWeight SINGLE 否 线宽，默认为：1
Transpatecy SINGLE 否 透明度，值域范围：0 -----100;默认为：100

数据完整性说明：

线段标识是主要标识；

图层名称指定线段所在图层（业务系统维护）；

流转方式：

由业务系统产生，不在其他系统中流转。

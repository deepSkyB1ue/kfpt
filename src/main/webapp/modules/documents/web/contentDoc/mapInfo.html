数据名称：

GIS地图中图层信息。

数据类型代码：

AddLayer、RemoveLayer、ClearLayer、HiddenLayer、LayersInfo

数据定义：

图层标识、图层名称、显示状态。

数据格式：

新增图层：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="AddLayer">

<LName Type = "Text">图层名</LName>

<LType Type = "Text">图层类型（Point、Line、Polygon）</LName>

…（数据可重复多次出现）

</Document<

注销图层：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="RemoveLayer">

<LName Type = "Text">图层名</LName>

…（数据可重复多次出现）

</Document>

清空图层：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="ClearLayer">

<LName Type = "Text">图层名</LName>

…（数据可重复多次出现）

</Document>

隐藏显示图层：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="HiddenLayer">

<LayerInfo>

<LName Type = "Text">图层名</LName>

<Hidden Type =”TEXT">隐藏显示图层（0：隐藏；1：显示）</Hidden>

</LayerInfo>

…（数据可重复多次出现）

<</Document>

获得图层信息：

<Document TaskGuid="组件标识" DataGuid="图层标识" DataType="LayersInfo">

<LayerInfo>

<LName Type = "Text">图层名</LName>

<Hidden Type =”TEXT">隐藏显示图层（0：隐藏；1：显示）</Hidden>

</LayerInfo>

…（数据可重复多次出现）

<</Document>

数据描述：

节点名称 类型 是否必填 含义
ID TEXT 是 图层唯一标识
Name TEXT 是 图层唯一标识
Name TEXT 是 图层名称
Hidden TEXT 否 是否显示图层

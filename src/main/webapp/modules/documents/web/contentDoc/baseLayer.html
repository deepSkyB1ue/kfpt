数据名称：

指定地图底图的服务路径，包括：地图服务路径设定。

数据类型代码：

MapServer

数据定义：

地图服务路径。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="MapServer">

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<MData>

<MName Type = "TEXT">szmap(地图服务名)</MName>

<MServer Type = "TEXT">http://gisserver.szjt.cn:8092/arcgis/services(服务地址)</MServer>

<MScaleLevel Type = "TEXT">>320000,160000,80000,40000,20000,10000,5000,2500(地图服务比例尺列表，BS可选)</MScaleLevel>

</MData>

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示面元素的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
MName TEXT 是 地图服务名称
MServer TEXT 是 服务地址
MScaleLevel TEXT 是 比例尺列表

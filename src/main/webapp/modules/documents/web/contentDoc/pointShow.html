数据名称：

点位数据信息，维护点位数据的基本信息，包括：点位标识、显示方式、经度、纬度、定位时间等。

数据类型代码：

DrawPoint

数据定义：

图层名称、自动缩放、自动居中、点位GUID、显示名称、经度、纬度、时间、速度、备注、图标ID、URL、Label显示。

数据格式：

<Document TaskGuid="组件标识" DataGuid="数据标识" DataType="DrawPoint">

<LName Type = "TEXT">用来显示点位的图层名称</LName>

<AutoZoom Type =”TEXT”>是否自动缩放1：是，0：否（可选，默认为1）</AutoZoom>>

<AutoPan Type =”TEXT”>是否自动居中1：是，0：否（可选，默认为1）</AutoPan>

<IsCluster Type =”TEXT”>是否点聚合1：是，0：否（可选，默认为1；BS专属参数）</IsCluster>

<PData>

<Guid Type=”GUID”>点位ID</Guid>

<Label Type = "TEXT">点位显示名称</Label>

<LON Type = "SINGLE">经度</LON>

<LAT Type = "SINGLE">纬度</LAT>

<Time Type = "DATE">点位时间（可选）</Time>

<Speed Type = "DOUBLE">速度（可选）</Speed>

<Direction Type = "LONG">方向</Direction>

<Status Type=”TEXT”>状态</Status>

<Remark Type = "TEXT">备注（中文，可选）</Remark>

<Icon Type = "TEXT">图标ID或相对路径()</Icon>

<DetailURL Type = "TEXT" >返回业务系统显示点位详情的URL</DetailURL>

<IsLabel Type =”TEXT">是否以label形式显示点位名称，1：显示，0：不显示（可选，默认为0）</ IsLabel >

<ImageUrl>http://10.38.133.146:8080/xzeckf/test.png</ImageUrl>

</PData>

…（数据可重复多次出现）

</Document>

数据描述：

节点名称 类型 是否必填 含义
LName Text 是 用来显示点位的图层名称。
AutoZoom TEXT 否
设置是否自动缩放地图比例。
0：否；
1：是；
AutoPan TEXT 否
是否自动居中。
0：否；
1：是；
IsCluster TEXT 否
点位显示是否需要聚合（BS专属参数）
0：否；
1：是；
Guid GUID 是 点位信息的唯一标识（Guid）
Label TEXT 否 点位显示的名称
LON SINGLE 是 经度举例：“120.123456”
LAT SINGLE 是 纬度举例：“31.123456”
Time DATE 否 定位时间
Speed SINGLE 否 当前速度(Km/h)
Remark TEXT 否 备注信息
Icon TEXT 是 图标编号或相对路径（BS相对路径为图标的URL，CS相对路径为，相对于GIS组件所在目录中指定图片文件夹GIMapImage的路径）
DetailURL TEXT 否 点位详细信息URL（由业务系统提供）
IsLabel TEXT 是
是否以Label显示点位
0：不显示；
1：显示；
ImageUrl TEXT 否 显示URL中的图片

数据完整性说明：

点位标识是主要标识

图层名称指定点所在图层（业务系统维护）；

流转方式：

车辆定时回传连续数据。
